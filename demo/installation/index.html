# Basic

Download the repo, extract vue-modal.umd.min.js and vue-modal.css out of the the dist folder and insert them in your page.

<script type="text/javascript" src="vue-modal.umd.min.js"></script>

<link rel="stylesheet" href="vue-modal.css">

# Module System

Install it via npm

npm i @kouts/vue-modal --save

import to use it

import VueModal from '@kouts/vue-modal';
import '@kouts/vue-modal/dist/vue-modal.css';
